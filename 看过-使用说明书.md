# 看过 — 使用说明书

## 一、快速开始

1. 双击 `seen.exe` 启动软件
2. 首次启动会提示选择数据存储位置，选一个文件夹即可
3. 之后启动直接进入主界面

软件为绿色便携版，不写注册表、不创建系统目录。整个文件夹拷贝到另一台电脑也能直接用。

---

## 二、界面布局

```
┌──────────┬─────────────────────────────────┐
│          │  搜索框      [调色板] [🌙] [📁] [⚙] │
│  分类1   ├─────────────────────────────────┤
│  分类2   │                                 │
│  分类3   │       封面卡片网格区域            │
│          │                                 │
│ + 添加   │                                 │
└──────────┴─────────────────────────────────┘
```

- **左侧**：分类列表，点击切换
- **顶部**：搜索框 + 工具栏按钮
- **中央**：当前分类下的作品封面网格

## 三、工具栏按钮

| 图标 | 功能 |
|------|------|
| 搜索框 | 按作品名称实时过滤 |
| 调色板 | 切换主题（素白 / 松绿 / 暖黄） |
| 月亮/太阳 | 切换亮色 / 暗色模式 |
| 文件夹 | 在资源管理器中打开数据文件夹 |
| 齿轮 | 更换数据存储位置 |

## 四、分类管理

### 切换分类
点击左侧分类名称即可切换，每个分类独立管理自己的作品。

### 添加分类
点击左下角「+ 添加分类」按钮，输入名称后回车。

### 重命名分类
鼠标悬浮在分类名称上，点击右侧出现的铅笔图标，输入新名称后回车。

### 删除分类
鼠标悬浮在分类名称上，点击垃圾桶图标，再次点击确认删除。**至少保留一个分类。**

### 拖拽排序
按住分类名称区域上下拖动，可调整分类顺序。顺序自动保存。

---

## 五、作品管理

### 添加作品
1. 点击网格首位的「+」虚线方框
2. 弹出添加窗口
3. **封面（可选）**：点击左侧封面区域，从电脑中选择一张图片
4. **作品名称（必填）**：输入名称
5. **观看日期（可选）**：自由填写
6. 点击「确认添加」

如果没有选择封面，系统会自动用竖排文字显示作品名称作为封面。

### 查看和编辑作品
点击任意作品封面，弹出详细浮窗：

- **左侧**：封面大图（点击可更换）+ 作品名称 + 观看日期
- **右侧**：Markdown 笔记编辑器

### 编辑笔记
- **编辑模式**：直接输入 Markdown 内容，停止输入 1 秒后自动保存
- **预览模式**：查看渲染后的效果
- 关闭浮窗时未保存的内容会自动写入磁盘

### 笔记中插入图片
1. 将图片文件放入数据文件夹对应分类的 `notes/` 目录下
2. 在笔记中写 `![](图片文件名.jpg)` 即可引用
3. 也支持外部链接：`![](https://example.com/image.jpg)`
4. 控制图片大小：`<img src="图片.jpg" width="300" />`

### 删除作品
鼠标悬浮在作品卡片上，点击右上角红色删除按钮，再次点击确认删除。

### 拖拽排序
鼠标悬浮在作品卡片上，按住右下角拖拽手柄，拖动到目标位置后松手。顺序自动保存。

### 笔记标记
有笔记内容的作品卡片底部会显示一条黄色短线，方便快速区分已记录和空白的作品。

### 搜索作品
在顶部搜索框输入关键词，实时过滤当前分类下的作品。

## 六、主题设置

点击顶栏调色板图标，可选择三种配色：

| 主题 | 说明 |
|------|------|
| 素白 | 黑白极简，经典百搭 |
| 松绿 | 绿意满满，护眼舒适 |
| 暖黄 | 温暖文艺，不刺眼 |

每种主题都有独立的亮色和暗色模式，点击月亮/太阳图标切换。

---

## 七、数据存储与备份

### 文件结构
```
你选的文件夹/
├── categories.json          # 分类列表
├── {分类名}/
│   ├── index.json           # 作品索引
│   ├── covers/              # 封面图片
│   │   └── {作品名}.jpg
│   └── notes/               # Markdown 笔记
│       └── {作品名}.md
└── ...
```

### 备份
直接复制数据文件夹即可备份所有数据。

### 迁移到新电脑
1. 复制 `seen.exe` 到新电脑
2. 复制数据文件夹到新电脑
3. 启动软件，点击齿轮图标重新选择数据文件夹位置

---

## 八、数据与文件系统同步

- 软件内所有操作（添加、删除、重命名）会**立即写入磁盘**
- 从其他程序切回本软件时，会**自动从磁盘读取**最新数据
- 可以直接在资源管理器中管理文件，切回软件即可看到变化

## 十、常见问题

**Q: exe 文件可以放在 U 盘里用吗？**
A: 可以，完全便携，不依赖任何系统路径。

**Q: 封面图片需要保留原文件吗？**
A: 不需要。选择封面后图片会自动复制到数据文件夹中，原始文件可以删除。

**Q: 笔记支持哪些 Markdown 语法？**
A: 标题、加粗、斜体、列表、引用、代码块、表格、链接、图片等标准语法均支持。

**Q: 怎么换图标？**
A: 准备一张 1024×1024 的 PNG 图片，运行 `npx tauri icon 你的图片.png` 即可生成所有尺寸的应用图标。
